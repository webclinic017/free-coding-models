# Changelog v0.3.74 - 2026-05-28

### Added
- **Caveman Code** tool mode — press `Z` to cycle to 🪨 Caveman Code, or launch directly with `--caveman`. Caveman Code is a token-efficient coding agent that wraps 20+ providers (Claude, ChatGPT, Copilot, Gemini, etc.) with ~2× fewer tokens than standard CLIs. Install with `npm install -g @juliusbrussee/caveman-code`, then authenticate via `caveman /login`.
- Auto-install detection for Caveman Code binary — FCM offers one-click install when the tool is missing
- Caveman Code entry in ⚡️ Command Palette (`Ctrl+P`)
